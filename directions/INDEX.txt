🔑 KEY CONCEPTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Model Accuracy (0.52):
  The model correctly predicts 52% of match outcomes.
  Better than random guessing (33%).

Match Probabilities:
  Example: Brazil 65%, Draw 18%, Germany 17%
  Shows likelihood of each outcome. Always sums to 100%.

Championship Probabilities:
  Based on 10,000 tournament simulations.
  Example: France 32% = France wins 3,200/10,000 simulations
  Shows tournament favorites.

Error: "No module named 'dotenv'"
  → Solution: uv pip install python-dotenv

Error: "No such file: data/raw/matches.csv"
  → Solution: python main.py prepare-data

Error: "API request failed"
  → Solution: Check .env file has valid API_KEY

Training is slow (2-5 minutes)
  → This is NORMAL - progress shown on terminal
